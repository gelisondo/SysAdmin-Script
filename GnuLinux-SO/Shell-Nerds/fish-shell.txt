Fish Shell:

Esta buena para trabajar con GIT, resarta las carpetas master o fork.
Es amigable con el usuario muy buen promp.
-resalta sintaxis
-auto sugerencias.
-auto completado con "Tab"


Fisher: plugin manager for fish
shellder: Shell theme
Nerd fonts: Powerline-patched font, SourceCode Pro.
Z: Directory jumping

peco: interactive filtering tool
-ctrl-r Busca comandos en el history
-ctrl-f Busca en los directorios de configuracion

Exa: Un ls mas moderno.

ghq: Local Git repository organizer.
Como no encontramos la forma de instalacion nata para debian nos bajamos el vinario.
  https://github.com/x-motemen/ghq
  https://github.com/x-motemen/ghq/releases
Al descomprimirlo lo movimos para /usr/local/bin



Configuración de alias:
#nano  /home/tokio/.config/fish/config.fish
if status is-interactive
	alias ll "exa -l -g --icons"
	alias llt='ll --tree --level=2 -a'
end
